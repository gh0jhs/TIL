# 2. Magic Method

```
__init__ : 생성자
__del__ : 소멸자
__str__ : 해당 객체의 출력 형태를 지정
__repr__ : 해당 객체의 출력될 수 있는 표현을 문자열의 형태로 반환?
```



# 3. Instance Method

```python
string.replace(old, new) : old값을 new값으로 교체한 스트링을 반환한다.
list.sort() : 리스트를 정렬해준다.
dict.items() : dictionary의 key값과 value값을 가져온다.
```
